# Pilot Study 项目文件夹

这是本次 Pilot Study 的项目文件夹。请在这个文件夹里完成任务，任务结束后保留整个文件夹。

## 文件夹说明

```text
P01_pilot_project/
├── 任务材料/             # 任务书、用户访谈材料、阶段输出表
├── 网页原型/             # 你要制作的网页小游戏
├── Codex对话记录/        # 研究者后续整理 Codex 对话
└── README.md
```

## 你主要要做什么

1. 阅读 `任务材料/02_学生任务书.doc`。
2. 阅读 `任务材料/03_用户访谈材料.doc`。
3. 在 `任务材料/04_阶段输出表.xls` 中填写五个阶段的结果。
4. 如有需要，阅读 `任务材料/05_Codex项目操作卡.doc`。
5. 在 `网页原型/` 中完成一个简单能玩的网页原型。
6. 任务结束后保留整个文件夹。

`.doc` 文件可以用 Word 或 WPS 打开，`.xls` 文件可以用 Excel 或 WPS 表格打开。`.md` 文件是给 Codex 阅读和备份用的。你自己阅读和填写时，优先使用 Word 和 Excel 文件。

你不需要一开始看懂 `网页原型` 里的技术文件。可以让 Codex 帮你阅读、修改和解释。制作出的网页小游戏会保存在 `网页原型` 文件夹里。

请不要删除 `任务材料/` 或 `Codex对话记录/`。
