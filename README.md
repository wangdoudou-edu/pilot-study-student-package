# Pilot Study 学生材料包

这是给学生使用的任务包。里面只包含完成任务需要的材料，不包含研究者访谈、观察、编码或内部研究说明。

## 第一步：拿到材料包

研究者会给你一个下载链接。下载后，请把材料包放到一个容易找到的位置，例如桌面。

如果你是在本地文件夹中阅读，请优先打开 `使用说明.doc`。

## 第二步：生成你的任务文件夹

研究者会给你一个编号，例如 `P01`。

打开终端，进入这个材料包所在的位置，然后运行：

```bash
bash setup_pilot_project.sh P01
```

运行后会生成一个新的文件夹：

```text
P01_任务文件夹/
```

如果你不熟悉终端，研究者会帮你完成这一步。

## 第三步：在 Codex 中打开任务文件夹

请在 Codex 中打开刚刚生成的 `P01_任务文件夹`。

打开后，请先输入：

```text
你好，我的编号是 P01，现在我要开始今天的任务了。请先阅读当前项目文件夹，告诉我里面有哪些文件。
```

请把 `P01` 换成你的编号。

## 第四步：完成任务

请优先打开这些文件：

- `任务材料/02_学生任务书.doc`
- `任务材料/03_用户访谈材料.doc`
- `任务材料/04_阶段输出表.xls`
- `任务材料/05_Codex项目操作卡.doc`

其中 `04_阶段输出表.xls` 是你需要填写的文件。

你可以直接用自然语言和 Codex 沟通，不需要先学代码。

`.doc` 文件可以用 Word 或 WPS 打开，`.xls` 文件可以用 Excel 或 WPS 表格打开。

## 第五步：结束任务

任务结束时，请在 Codex 中输入：

```text
今天的任务到此为止。请不要再修改文件，请帮我简单总结一下你今天帮我做过哪些事情。
```

之后请保留整个 `P01_任务文件夹`。
